# 📊 Interactive Sales & Profit Power BI Dashboard

A comprehensive, interactive Power BI dashboard designed to provide end-to-end business intelligence across sales, products, time, and geography.

## 📁 Dashboard File

> 🔗 https://github.com/Sreeram-Pranavi/powerbi-sales-profit-dashboard/blob/main/Power%20Sales%20Analysis.pbix

## Dashboard Modules

### 1. Executive Sales & Profit Overview
High-level KPIs including total revenue, gross profit, profit margin, and YoY growth — built for C-suite and leadership decision-making.

### 2. Product Deep Dive
Granular analysis of product-level performance: top/bottom performing SKUs, category contribution, and profitability breakdown.

### 3. Time & Performance Drill-Down
Trend analysis across daily, monthly, quarterly, and yearly dimensions with dynamic drill-through capabilities.

### 4. Region & Market Analysis
Geographic sales distribution, market share by region, and territory-level performance comparisons.

## Features
- Fully interactive slicers and cross-filtering
- Drill-through and drill-down navigation
- Dynamic tooltips and KPI cards
- Responsive layout for both desktop and presentation mode

## Tools Used
- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Language)
